<h2 align="center">Bank app</h2>

Bank on  Django Rest Framework.

### Инструменты разработки

**stack:**
- Python >= 3.8
- Django Rest Framework
- Postgres

## Start

#### 1) Create image

    docker-compose build

##### 2) Run container

    docker-compose up
    
##### 3) Go to address

    http://127.0.0.1:8000/api/v1/swagger/
